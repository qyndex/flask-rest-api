# Flask REST API

Minimal Flask 3 REST API using the create_app factory, SQLAlchemy 2 models, marshmallow schemas, Alembic migrations, and Blueprint-based routing.

**Stack:** Flask | **Complexity:** simple
**Tags:** `flask`, `rest`, `api`, `python`, `sqlalchemy`, `marshmallow`, `backend`

## Quick Start

```bash
pip install -r requirements.txt
flask run --reload
```

## Docker

```bash
docker compose up --build
```
